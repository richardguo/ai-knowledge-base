## 摘要
Claude Context 是一个基于 Model Context Protocol (MCP) 的 TypeScript 项目，由 Zilliz 团队开发。它通过语义代码搜索技术，将整个代码库作为上下文提供给 Claude Code 及其他 AI 编程助手，解决了大型代码库在 AI 辅助编程中因上下文窗口限制导致的成本高昂和效率低下问题。项目利用向量数据库高效存储和检索代码片段，仅将相关代码注入 AI 上下文，显著降低了 token 使用成本。它支持与多种 AI 编码代理（如 Claude Code）集成，并提供了 VS Code 扩展和 npm 包，便于开发者快速部署和使用。

## 核心亮点
- 利用语义搜索实现全代码库上下文检索
- 通过向量数据库显著降低大型代码库的 AI 使用成本
- 基于 MCP 协议，可灵活集成多种 AI 编码助手
