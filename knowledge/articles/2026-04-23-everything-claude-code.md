## 摘要
Everything Claude Code 是一个面向AI代理性能优化的综合性系统，由Anthropic黑客马拉松获奖者开发。项目旨在为Claude Code、Codex、Cursor、OpenCode等AI代理平台提供技能、本能、内存优化、持续学习、安全扫描和研究优先的开发能力。它不仅包含配置，还提供完整的生产就绪代理、技能、钩子、规则、MCP配置以及经过10个月以上实际产品构建验证的遗留命令桥接。项目支持多语言生态系统（包括TypeScript、Python、Go、Java等），拥有超过14万星标、2.1万分支和170多位贡献者，体现出强大的社区影响力和成熟度。
