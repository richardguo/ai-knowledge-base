## 摘要
oh-my-openagent 是一个基于 TypeScript 的多模型 AI Agent 编排框架，旨在打破单一供应商的锁定。它允许开发者混合使用 Claude、GPT、Gemini 等多种大模型，利用各自在推理、速度或创意上的优势进行任务协作。项目提供 TUI 界面，作为 Claude Code 的开源替代方案，深度集成 IDE 工作流。它强调开放性与灵活性，致力于构建一个不依赖特定生态的 Agent 驱动开发环境。

## 核心亮点
- 多模型编排架构，支持混合使用 Claude、GPT、Gemini 等模型，打破单一供应商锁定
- 提供 TUI 界面，定位为 Claude Code 的开源替代品，深度集成开发工作流
- 高度活跃的社区，采用实时公开构建模式，热度极高，引发广泛关注
