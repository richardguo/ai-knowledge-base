## 摘要
Claude Context 是一个基于 MCP（Model Context Protocol）的语义代码搜索插件，专为 Claude Code 和其他 AI 编程助手设计。它通过将整个代码库嵌入向量数据库，实现高效的语义检索，仅将最相关的代码片段提供给 AI 作为上下文，从而大幅降低大代码库场景下的 token 使用成本。该项目由 Zilliz 团队开发，提供核心库、MCP 服务端和 VS Code 扩展，支持多种嵌入模型和向量数据库集成，旨在为 AI 编程代理提供持久、精准的代码库上下文记忆，提升代码理解和生成的效率。

## 核心亮点
- 利用向量数据库实现整个代码库的语义搜索，为AI编程助手提供深度上下文
- 通过MCP协议集成，支持Claude Code等多种AI编码代理，成本效益高
- 提供完整的工具链，包括核心库、MCP服务和VS Code扩展，易于部署使用
