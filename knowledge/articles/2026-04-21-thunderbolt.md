## 摘要
Thunderbolt 是一款开源、跨平台的 AI 客户端，旨在让用户完全掌控 AI 模型和数据，消除厂商锁定。它支持所有主流桌面和移动平台（Web、iOS、Android、Mac、Linux、Windows），兼容前沿、本地和本地部署的模型。项目基于 TypeScript 开发，采用 Tauri 框架实现跨平台桌面应用，前端使用 React 和 Vite，后端可选部署为 Docker 容器或 Kubernetes。当前处于企业级生产准备阶段，正在进行安全审计，主要面向需要本地化部署的企业客户。用户需自行配置模型提供商（如 Ollama、llama.cpp 或 OpenAI 兼容的 API），支持离线优先目标但部分功能（如认证、搜索）仍依赖后端服务。

## 核心亮点
- 跨平台支持所有主流桌面和移动操作系统
- 开源且支持本地化部署，消除厂商锁定
- 兼容多种模型提供商，包括本地推理和云端 API
