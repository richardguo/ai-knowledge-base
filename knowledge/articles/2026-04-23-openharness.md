## 摘要
OpenHarness 是一个由香港大学数据科学实验室（HKUDS）开发的轻量级开源智能体框架，提供核心的智能体基础设施，包括工具调用、技能、记忆和多智能体协作能力。其内置的个人智能体 ohmo 基于该框架构建，支持在飞书、Slack、Telegram、Discord 等平台上进行长会话协作，能够自主完成代码分支管理、编写、测试和提交拉取请求等开发任务。项目强调无需额外API密钥，可直接利用用户已有的 Claude Code 或 Codex 订阅。框架采用 Python (≥3.10) 开发，包含超过 43 种工具，通过了 114 项测试，并提供了 CLI 集成支持，如 OpenClaw、nanobot 等。

## 核心亮点
- 提供核心轻量级智能体基础设施（工具、技能、记忆、多智能体协调）
- 内置个人智能体 ohmo 可在主流协作平台进行长会话并自主执行开发任务
- 无需额外API密钥，直接利用现有 Claude Code 或 Codex 订阅运行
